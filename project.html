OFDM Simulation Project

Implemented by: MATLAB. [Source Code]
Time: May 2017 - June 2017.

Implemented orthogonal frequency-division multiplexing (OFDM) system in Matlab.
Designed effective IIR filter to do signal processing on transceiver for OFDM symbol.
Used BPSK/QPSK modulation, FFT/IFFT for OFDM symbol, cyclic prefix (CP), up/down-sampling. The system's BER is consistent (close) to the theoretical one.
